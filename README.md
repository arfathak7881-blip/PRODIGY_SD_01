# Temperature Conversion Program

## Description
A Java GUI application that converts temperatures between Celsius, Fahrenheit, and Kelvin.

## Features
- Celsius to Fahrenheit and Kelvin
- Fahrenheit to Celsius and Kelvin
- Kelvin to Celsius and Fahrenheit
- User-friendly GUI
- Input validation

## Technologies Used
- Java
- Swing
